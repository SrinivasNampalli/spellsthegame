## üìñ About The Project

**Spells the Adventure** is a top-down action-adventure RPG where you play as a young mage exploring a mystical world filled with magic, danger, and discovery. Master the art of spellcasting by combining elemental forces to create devastating attacks, solve environmental puzzles, and defeat powerful enemies.

### Why This Game Rocks:
* **Dynamic Spell Crafting** - Combine 6 base elements to create 15+ unique spell combinations
* **Strategic Combat** - Elemental weaknesses and combo systems reward smart play
* **Environmental Magic** - Your spells interact with the world (freeze water, burn obstacles, etc.)
* **Roguelite Elements** - Randomized dungeons with persistent character progression
* **Rich World** - Explore 4 distinct biomes, each with unique challenges and enemies

**Target Playtime:** 8-12 hours for first completion
**Development Time:** 200+ hours over 30 days

## üéØ Gameplay

### Core Loop
1. **Explore** diverse environments and discover secrets
2. **Collect** spell components and magical ingredients
3. **Craft** powerful spell combinations
4. **Battle** enemies using strategic elemental magic
5. **Progress** through skill trees and unlock new abilities

### Combat Philosophy
Combat in Spells the Adventure emphasizes **preparation and execution**:
- Before fights: Equip spells based on enemy types
- During fights: Manage mana, dodge attacks, exploit weaknesses
- After fights: Collect loot and resources

### Difficulty & Balance
- **3 Difficulty Settings:** Casual, Adventurer, Archmage
- **Fair Challenge:** Enemies telegraph attacks
- **No Grinding:** Skill matters more than stats
- **Roguelite Safety:** Persistent progression even on death

## üíæ Installation

### Playing the Game

#### Online (Easiest)
1. Visit [spellstheadventure.io](#) or [itch.io page](#)
2. Click **Play in Browser**
3. Start your adventure!

#### Local Download
1. **Download** the latest release
2. **Extract** the ZIP file
3. **Open** `index.html` in any modern browser
4. **Enjoy!**

### Development Setup

1. Clone the repository
   ```sh
   git clone https://github.com/yourusername/spells-the-adventure.git
   ```
2. Open the project folder
   ```sh
   cd spells-the-adventure
   ```
3. Start a local server (choose one):
   ```sh
   # Python 3
   python -m http.server 8000

   # Node.js (with http-server)
   npx http-server

   # PHP
   php -S localhost:8000
   ```
4. Open browser and navigate to `http://localhost:8000`
5. Start developing!

### System Requirements

**Minimum:**
- Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- 2 GB RAM
- JavaScript enabled
- 50 MB storage for save data

**Recommended:**
- Latest Chrome or Firefox
- 4 GB RAM
- Hardware acceleration enabled
- Fast internet connection (for online play)
